# real-world-vue

## Stack

- Vue(Vue router, Vuex)
- Axios: API call
- JSON server: Mock DB
- NProgress: Progress Bar

## Project setup

```
yarn install
```

### Start a mock API server(JSON Server)

```
yarn db
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
